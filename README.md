# 메타인지 팩폭 분석 도구

사주팔자, 서양 점성술, 자미두수, 조티샤 — 4개 동서양 운명 체계를 교차 분석하여 자기기만 구조, 인지적 사각지대, 반복 패턴의 맹점을 해부하는 메타인지 분석 도구.

**운세를 보지 않습니다.** "앞으로 잘 될 거다"가 아니라, "당신이 왜 매번 같은 구덩이에 빠지는지, 왜 그걸 스스로 못 보는지"를 네 체계의 교차 분석으로 해부합니다.

## 에이전트 구성

| 에이전트 | 체계 | 집중 영역 |
|----------|------|----------|
| 거울이 | 오케스트레이터 | 4체계 교차 비교 + 통합 리포트 |
| 방울이 | 사주팔자 (명리학) | 반복하는 선택 패턴 |
| 별이 | 서양 점성술 | 자기 이미지 왜곡 |
| 칠성이 | 자미두수 | 회피하는 삶의 영역 |
| नम (나마) | 조티샤 (베딕 점성술) | 자기 합리화 구조 |

## 핵심 설계 원칙

- **독립 컨텍스트:** 4개 에이전트가 서로의 결과를 모른 채 병렬 분석. 같은 맹점을 독립적으로 지목하면 신뢰도 상승.
- **1회용 도구:** 데이터를 저장하지 않음. 분석 후 팔로업 카드만 발급.
- **팩폭 ≠ 인격 훼손:** 불편한 진실을 직면시키되, 최종 목표는 행동 변화.

## 사용 방법

1. Claude Code에서 이 프로젝트를 열기
2. `@거울이` 호출
3. 생년월일, 태어난 시간, 성별, 출생지 입력
4. 팩폭 강도 선택: (a) 풀파워 / (b) 적당히
5. 분석 대기 (약 10분)

## 요구 사항

- Claude Code 구독 (Claude Max 권장)
- Python 3.x (계산기 실행용)
